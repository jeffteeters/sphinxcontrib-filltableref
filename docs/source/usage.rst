Usage
=====

As described in :ref:`about`, the extension works using two directives:  “tblrender” - specifies what a
summary table looks like (but without referenced values), and the 
“tbldata” directive specifies the values used to fill in the summary table) along with citations to publications.

These directives are described in the following sections:

   tbldata
